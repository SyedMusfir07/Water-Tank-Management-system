# Smart Water Quality Monitoring System (SWQMS)

ESP32-based IoT Water Quality Monitoring System using:

- TDS Sensor (ppm)
- Turbidity Sensor (NTU)
- DS18B20 Temperature Sensor (°C)

## Features
- Real-time monitoring
- Firebase Realtime Database
- Web dashboard (HTML/CSS/JavaScript)
- Pollution alert system

## Project Structure
- firmware/esp32_swqms → ESP32 Arduino code
- web-dashboard → Web dashboard files
